# EECS 2031 Practice

This repository contains practice problems and coding exercises for **EECS 2031 (Software Tools)** at York University.

## Structure
- `tester.c`: A practice file to experiment with C programming concepts, arrays, functions, and pointers.

## How to Compile and Run
To compile and run any of the C files, you can use the `gcc` compiler in your terminal:

```bash
# Compile the code
gcc filename.c -o output_name

# Run the executable
./output_name
```

*(Note: The compiled executable files are ignored by git so they aren't uploaded to the repository).*
